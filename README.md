# react-bdd-sample
bdd sample for react using react-testing-library

# Requirement

```
node v12.13.1
yarn 1.19.2
```

# Usage

```
# boot App
yarn install
yarn start

# lint
yarn lint
yarn auto_fix

# exec test
yarn test

# build App
yarn build
```
